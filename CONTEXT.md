# Word Learning

This context describes the vocabulary learning product language shared by classroom organization, learning
resources, and student study workflows.

## Language

**班级**:
由一名班级负责老师负责的教学组织，包含当前班级学生成员，并作为学习资源、学习计划和班级群消息流的组织边界。
_Avoid_: 学习组, 群组

**班级名称**:
用于识别班级的全局唯一名称。任何两个班级都不能使用相同名称，即使班级负责老师不同或其中一个班级已归档。
_Avoid_: 同教师下唯一名称, 可重复班名

**班级负责老师**:
当前对班级承担管理责任的教师。一个班级在同一时间只有一名班级负责老师。
_Avoid_: 班主任, owner, classroom owner

**班级转移**:
管理员将班级当前负责老师变更为另一名教师的管理动作。班级转移只改变未来班级管理权，不改变已发布学习计划的归属老师或历史统计归属。
_Avoid_: 计划转移, 历史迁移

**班级学生成员**:
当前加入某个班级的学生。成员关系决定学生是否处在该班级的消息、资源和学习安排范围内。
_Avoid_: 群成员, 班级用户

**归档班级**:
不再用于新增成员、发布新资源、创建新学习计划或访问当前班级群消息流的历史班级。归档班级保留学习计划、学习记录、统计和消息等历史追溯价值。
_Avoid_: 删除班级, 停用班级

**班级群消息流**:
由班级自动拥有的师生消息空间，以消息流形式呈现文字留言和学习资源发布。它的可见范围由当前班级负责老师和当前班级学生成员决定。
_Avoid_: 班级群, 群空间, 群聊

**教学视频**:
用于课堂讲解、学习安排或班级群消息流发布的视频学习资源。教学视频可以来自老师上传、管理员上传或云端空间同步。
_Avoid_: 普通视频, 云视频, 媒资

**公共教学视频**:
已经由管理员或有权限的管理者确认可用于教学分发的视频。公共教学视频对全体老师可见、可预览、可发布到自己负责的班级群，但不会让学生直接浏览整个视频库。
_Avoid_: 全员公开视频, 系统公开视频, 老师私有视频

**班级群视频入口**:
班级负责老师把公共教学视频发布到班级群消息流后形成的视频学习入口。当前班级学生成员可以通过该入口播放视频，入口不会授予浏览视频库或管理视频的权限。
_Avoid_: 视频授权表, 视频分配, 全局播放权限

**学生视频播放授权**:
学生播放教学视频的业务许可。它可以来自学生可见的公开视频范围，也可以来自当前班级群视频入口，并且播放时仍受当前班级成员关系和视频可用状态约束。
_Avoid_: 老师上传人权限, 视频所有者权限, 永久播放权限

**学习计划学生快照**:
学习计划发布时由目标班级当前学生成员确定的学生集合。发布后班级成员变化不会自动改变该集合，新学生需要通过显式追加进入已发布学习计划。
_Avoid_: 自动同步学生, 动态班级计划

**离班计划学生**:
曾属于学习计划目标班级并已获得个人学习计划，后来被移出班级的学生。离班计划学生可以继续完成已分配给自己的旧学习计划，但不再获得该班级的新资源、新消息或新计划。
_Avoid_: 已退班学生, 删除学生计划

**计划历史报表**:
围绕某个已发布学习计划展示的历史和后续完成情况报表。计划历史报表包含学习计划学生快照中的学生，即使学生后来离开班级。
_Avoid_: 当前班级报表, 班级现状统计

**当前班级统计**:
以当前班级学生成员为统计对象的班级运营数据。离班计划学生不计入当前班级统计。
_Avoid_: 计划历史统计, 历史班级统计

**已发布学习计划**:
已经下发给学习计划学生快照的学习安排。发布后班级成员或班级辞书分配变化不会回写其既有学生计划、学习记录和学习进度。
_Avoid_: 动态计划, 当前班级计划

**学习计划归属老师**:
创建并发布学习计划时确定的负责教师。班级后续转移给其他老师时，既有学习计划的归属老师不随班级变更。
_Avoid_: 当前班级老师, 接手老师
